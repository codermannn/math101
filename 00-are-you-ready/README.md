# 🧪 Prerequisites Check

> *Know your starting point before the journey.*

---

## What This Tests

The diagnostic notebook checks two things:

1. **Python fundamentals** — list comprehensions, recursion, dictionaries, loops, slicing
2. **Math fundamentals** — logarithms, summations, exponents, factorials, growth intuition

## How to Use

1. Open `DIAGNOSTIC.ipynb`
2. Replace every `___` with your answer **before running the cell**
3. Run each cell to check your answer
4. Review your score at the end

## Scoring

| Score | Readiness |
|-------|-----------|
| 90–100% | ✅ Ready — proceed to `TOOLS_PRIMER/` |
| 70–89% | ⚠️ Proceed with caution, review weak spots |
| Below 70% | 📚 Strengthen fundamentals first |
